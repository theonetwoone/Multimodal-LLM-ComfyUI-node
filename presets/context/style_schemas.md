# Style schemas (reference block)

Use these as reusable "style blocks" you can paste into prompts or keep in `extra_context`.

## Photoreal
- Medium: photography, photoreal, natural texture
- Lighting: realistic, soft key + fill, believable shadows
- Camera: 35mm/50mm, shallow depth of field (optional)
- Constraints: no plastic skin, no CGI look

## Cinematic
- Mood: cinematic, dramatic contrast, filmic color grade
- Lighting: rim light, volumetric haze (optional)
- Camera: anamorphic, wide establishing / tight close-up (specify)
- Details: bokeh, film grain (subtle)

## Anime / Illustration
- Medium: anime illustration, clean linework
- Shading: cel shading / soft shading (pick one)
- Color: vibrant palette, controlled highlights
- Constraints: consistent character design, clean anatomy

## Comic / Graphic Novel
- Medium: inked comic, bold outlines
- Shading: crosshatch / halftone (pick one)
- Color: limited palette (optional)
- Composition: dynamic angles, strong silhouettes

## 3D / CGI
- Medium: 3D render, PBR materials
- Lighting: studio softbox / HDRI (specify)
- Camera: realistic lens + DOF (optional)
- Constraints: avoid uncanny skin, avoid low-poly artifacts

## Pixel Art
- Medium: pixel art, limited palette
- Resolution: specify (e.g. 64×64, 128×128)
- Constraints: crisp pixels, no anti-aliasing, no gradients unless dithered

## Watercolor
- Medium: watercolor painting on textured paper
- Edges: soft bleeding, granulation
- Color: transparent washes, subtle blooms

## Oil Painting
- Medium: oil on canvas, visible brushstrokes
- Lighting: painterly highlights, rich shadows
- Color: warm/cool balance, layered paint

## Minimal / Flat
- Medium: flat vector, minimal shapes
- Color: 2–5 colors, clean spacing
- Constraints: no textures, no gradients (unless requested)

## UI / Product Mock
- Medium: UI screenshot / product render (specify)
- Layout: grid, spacing system, typography scale
- Constraints: readable text, consistent alignment
