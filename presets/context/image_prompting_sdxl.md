# SDXL prompting quick schema

Use this as a reference block when writing prompts for SDXL-style models.

- Subject: who/what, key attributes
- Scene: environment, time of day, weather
- Style: medium, artist/style tags, era
- Composition: framing, pose, perspective, rule of thirds, focal point
- Lighting: soft/hard, direction, color temperature
- Camera: lens, aperture, ISO (optional)
- Quality tags: high detail, sharp focus (use sparingly)

Optional:
- Negative prompt: artifacts to avoid (hands, text, extra limbs, blur, watermark)
