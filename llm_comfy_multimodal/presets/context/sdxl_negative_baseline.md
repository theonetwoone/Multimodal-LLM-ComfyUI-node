# SDXL negative baseline (short + targeted)

Use negatives as a guardrail, not a second full prompt.

## Baseline (general)
worst quality, low quality, blurry, jpeg artifacts, watermark, text, logo, signature

## Anatomy add-on (when needed)
bad anatomy, deformed, disfigured, extra limbs, extra fingers, fused fingers, missing fingers, deformed hands

## Face add-on (when needed)
deformed face, asymmetrical eyes, uncanny, plastic skin

## Composition add-on (when needed)
cropped, out of frame, duplicate subject

Rule of thumb: start with baseline, then add only 3–8 targeted negatives that match the failure you see.
